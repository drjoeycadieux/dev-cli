How It Works

Three simple steps to install any developer tool

1

Open PowerShell

Launch PowerShell on your Windows machine and navigate to your project directory.

→
2

Run the Command

Type dev-cli install [tool-name] (e.g., dev-cli install nodejs)

→
3

Done!

Dev-CLI automatically downloads and installs your tool via winget. No manual configuration needed.

Example Workflow
