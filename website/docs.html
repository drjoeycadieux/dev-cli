Introduction

Dev-CLI is a lightweight, high-performance command line interface designed specifically for Windows developers. It wraps the powerful Windows Package Manager (winget) and provides a curated experience with modern terminal aesthetics.

Built with 🦀 Rust, Dev-CLI ensures that your tool management is not only fast but safe and reliable.

Installation

To install Dev-CLI, run the following command in an administrative PowerShell window:

iwr -useb https://dev-cli.netlify.app/install.ps1 | iex

Verify the installation by running:

dev-cli --version

Basic Usage

The core philosophy of Dev-CLI is simplicity. Most operations can be done with a single command.

Installing a tool

To install a tool from our curated list, simply use:

dev-cli install nodejs

You can also install any package by its winget ID:

dev-cli install Microsoft.PowerShell

Commands Reference

dev-cli list     # Show all curated tools
dev-cli install  # Install a specific tool
dev-cli check    # Check system for installed tools
dev-cli --help   # Show help information
